# Smoke Testing Checklist

## Objetivo
Validar que las funcionalidades principales del e-commerce funcionen correctamente luego del deploy.

## Checklist

- [x] Registro de usuario
- [x] Login correcto
- [x] Validación de login inválido
- [x] Navegación entre categorías
- [x] Búsqueda de productos
- [x] Visualización de detalle de producto
- [x] Selección de talle
- [x] Agregado al carrito
- [x] Actualización de cantidades
- [x] Actualización del total
- [x] Eliminación de productos del carrito
